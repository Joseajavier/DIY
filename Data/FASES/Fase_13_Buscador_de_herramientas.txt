FASE 13 — BUSCADOR DE HERRAMIENTAS CON FILTROS Y CATALOGO DE MARCAS

OBJETIVO
Crear un buscador de herramientas dentro de la app que permita al usuario
encontrar la herramienta ideal segun sus necesidades, presupuesto y nivel
de uso. Con catalogo de marcas reales organizado por gamas.

FLUJO DEL USUARIO
1. Pulsa "Buscar herramientas" (nuevo boton en Home o tab)
2. Selecciona TIPO de herramienta (sierra circular, taladro, lijadora, etc.)
3. Filtra por:
   - Gama: Basica / Media / Profesional
   - Uso: Casa / Taller / Obra
   - Presupuesto: <50€ / 50-100€ / 100-200€ / 200-500€ / >500€
   - Alimentacion: Bateria / Cable / Manual
4. Ve un listado de herramientas reales organizadas por marca y gama
5. Cada herramienta muestra: marca, modelo, precio orientativo, para que sirve
6. Opcionalmente enlace de compra (afiliado Amazon cuando este disponible)

MODELO DE DATOS

ToolCategory
  - id: string
  - name: string (ej: "Corte")
  - icon: string

ToolType
  - id: string
  - categoryId: string
  - name: string (ej: "Sierra circular")
  - description: string
  - icon: string

ToolBrand
  - id: string
  - name: string (ej: "DeWalt")
  - country: string (ej: "USA")
  - tiers: ('basic' | 'mid' | 'pro')[]
  - logo?: string

ToolProduct
  - id: string
  - typeId: string
  - brandId: string
  - model: string (ej: "DWE575K")
  - tier: 'basic' | 'mid' | 'pro'
  - use: ('home' | 'workshop' | 'construction')[]
  - power: 'battery' | 'corded' | 'manual'
  - priceRange: { min: number, max: number }
  - description: string
  - features: string[]
  - bestFor: string (ej: "Cortes rectos en tableros de hasta 65mm")
  - amazonSearchQuery?: string (para buscar luego en API)

ToolFilter
  - type?: string
  - tier?: 'basic' | 'mid' | 'pro'
  - use?: 'home' | 'workshop' | 'construction'
  - priceMax?: number
  - power?: 'battery' | 'corded' | 'manual'
