### 🔴 #1 — `src/screens/ToolSearchScreen.tsx:70,74,76,78` + `WoodCatalogScreen.tsx:76,80,82,84` — TS2322 en Chip con `key`

**Descripción:** El componente `Chip` definido inline declara `{ label: string; active: boolean; onPress: () => void }`. Al usarlo en `.map()` se pasa `key` como prop, pero sin `@types/react` instalado TypeScript lo trata como una prop extra no permitida.

**Código ACTUAL (problema):**
```tsx
// ToolSearchScreen.tsx:58
const Chip = ({ label, active, onPress }: { label: string; active: boolean; onPress: () => void }) => (
  <TouchableOpacity ...>...</TouchableOpacity>
);

// Uso en línea 70 — TS2322
{TOOL_CATEGORIES.map(c => <Chip key={c.id} label={...} active={...} onPress={...} />)}
```

**Código DESPUÉS (fix):**
```tsx
// Añadir key?: React.Key al tipo del Chip
const Chip = ({ label, active, onPress }: { label: string; active: boolean; onPress: () => void; key?: React.Key }) => (
  <TouchableOpacity ...>...</TouchableOpacity>
);
```

### 🟡 #4 — `src/services/catalogApiClient.ts` — Tipos `any` en cache y respuesta

**Descripción:** Las variables de cache `toolsCache: any` y `woodCache: any` anulan el tipado de `CatalogResponse`/`WoodCatalogResponse`. Los arrays internos (`categories: any[]`, `types: any[]`, `products: any[]`) propagan `any` por todo el árbol. No causa errores TS pero puede causar crashes silenciosos en runtime si el backend devuelve formato inesperado.

**Código ACTUAL (problema):**
```tsx
// catalogApiClient.ts:7-8
let toolsCache: any = null;
let woodCache: any = null;

export interface CatalogResponse {
  version: number;
  lastUpdated: string;
  categories: any[];   // ← sin tipo
  types: any[];        // ← sin tipo
  brands: any[];       // ← sin tipo
  products: any[];     // ← sin tipo
  totalProducts: number;
}
```

**Código DESPUÉS (fix):**
```tsx
// Definir tipos concretos antes del cache
let toolsCache: CatalogResponse | null = null;
let woodCache: WoodCatalogResponse | null = null;

export interface CatalogResponse {
  version: number;
  lastUpdated: string;
  categories: { id: string; name: string; icon: string }[];
  types: { id: string; name: string; categoryId: string }[];
  brands: { id: string; name: string }[];
  products: import('../models/tool').ToolProduct[];
  totalProducts: number;
}
```

**Archivo:** `src/services/catalogApiClient.ts:7-8,13-20`
**Impacto:** Previene crashes silenciosos por respuestas inesperadas del backend
